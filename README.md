# ontology-of-connections
Гипотеза онтологии связей (ГОС) / Hypothesis of Ontology of Connections (HOC) / 连接本体论假说 (HOC). A unified model of reality, where primary are the connections between energy quanta that generate stable nested structures of "I ↔ We." Explains the origin of space-time, the laws of physics, consciousness, and social processes from a single logic.
