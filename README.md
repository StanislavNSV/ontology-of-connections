# Гипотеза онтологии связей (ГОС) / Hypothesis of Ontology of Connections (HOC)  /  连接本体论假说 (HOC)

## Основная суть гипотезы / The main essence of the hypothesis

| Первичны всего не частицы и не формы — первичны связи между квантами энергии. |
| ------------------------------------------------------------ |
| Из этих связей эмерджентно рождаются вложенные структуры «Я ↔ Мы», пространство-время, законы физики, сознание и социальные процессы. Это не ещё одна «теория всего», а минимальная онтология, снимающая парадоксы там, где отдельные науки заходят в тупик. |

| Particles and forms are not primary—the connections between energy quanta are primary. |
| ------------------------------------------------------------ |
| From these connections, nested structures of "I ↔ We," space-time, the laws of physics, consciousness, and all social processes emerge. This is not another "theory of everything," but a minimal ontology that resolves paradoxes where individual sciences reach a dead end. |
